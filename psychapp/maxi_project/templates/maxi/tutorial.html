Syllogism Tutorial

Please take the time necessary to understand the content of this short tutorial on solving categorical syllogisms.


A "categorical syllogism" is a type of argument that consists of three statements: two premises and one conclusion.

Each statement relates two categories, by asserting that either all or part of the first class is included or excluded from the other.
For categories A and B, this means that there are four different types of statement:


  • All A are B:There exists at least one element in A, and A is wholly included in B

  • Some A are B:There exists at least one element in A that is included in B

  • No A are B:There exists at least one element in A and one in B, and A is wholly excluded from B

  • Some A are not B:There exists at least one element in A that is excluded from B

These four different types ("All", "Some", "No" and "Some-not") can be visualised through "Euler diagrams":



In an Euler diagram, overlap means class inclusion and a cross means "There must exist an individual in this area for the statement to be true". For example, the "Some A are B" statement means that there exists at least one element inside A that is also in B, without there necessarily existing an individual in the other, non-overlapping regions.

In a syllogism, the first premise states something about two categories, which we may call A and B. The second premise states something about B and C, and the conclusion states something about A and C. Thus, the category labeled B occurs in both premises but not in the conclusion. For example:

All A are B

No C are B

-------------

Therefore, no C are A

Euler diagrams can be used to determine which conclusions are valid from two premises. They can be thought of as metallic hoops in which the "necessary individual" dot is a pin.

  1. Lay out hoops and pins to represent the premises.
  2. Resize one of the premises and move it so that the B-hoops align (they refer, after all, to the same term).
  3. Constrained by the pin, test all possible arrangements that are consistent with the premises and identify the one that creates as many subregions as possible (each region corresponds to a type of individual, and we don't want to forget any of them.
  4. If a hoop that is immediately surrounding a pin gets subdivided, remove that pin.

After this has been done, we can read off the diagram, to see which relationships hold for A and C, i.e. which conclusions are valid. Dots that remain represent individuals that have been specified with regard to all three categories, so if there are no dots remaining, we cannot conclude anything.

Below follow three examples of how to use Euler diagrams:


Premises: "All A are B" and "Some C are not B"

Conclusion: "Some C are not A"



Premises: "All B are A" and "All B are C"

Conclusion: "Some A are C"



Premises: "Some A are B and "Some C are B"

Conclusion: "No valid conclusion"

Euler diagrams can be used to determine which conclusions are valid from two premises.

  1. Make diagrams to represent the two premises.
  2. Make the B-rings correspond (they refer, after all, to the same term).
  3. Of all arrangements consistent with the initial diagrams, choose the one that creates the most subregions.
  4. If a minimal region with a dot (i.e. one closest to the dot) gets subdivided as a result, remove it.

After this has been done, we can read off the diagram to see which relationships hold for A and C, i.e. which conclusions are valid. Dots that remain represent individuals that have been specified with regard to all three categories, so if there are no dots remaining, we cannot conclude anything.
